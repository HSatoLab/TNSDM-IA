# TNSDM-IA

-This repository provides open-source MATLAB code for TNSDM-IA.

-The code is designed to work seamlessly with the ([PlatEMO](https://github.com/BIMK/PlatEMO)) platform for evolutionary multi-objective optimization.

## How to run

  1. Download [PlatEMO](https://github.com/BIMK/PlatEMO/releases).

  2. Copy the TNSDM-IA directory into "Algorithms/Multi-objective optimization".

  3. Run TNSDM-IA in the same manner as other algorithms in PlatEMO.
